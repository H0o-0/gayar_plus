# توثيق نظام التنقل (Navigation Bar) في مشروع Gayar Plus
# Documentation of Navigation System in Gayar Plus Project

## نظرة عامة على النظام
## System Overview

نظام التنقل في مشروع Gayar Plus يتكون من شريط تنقل علوي يحتوي على قائمة ملحقات (Accessories) منسدلة مع قوائم فرعية للعلامات التجارية والفئات والموديلات.

The navigation system in Gayar Plus project consists of a top navigation bar with a dropdown accessories menu that includes submenus for brands, categories, and models.

## هيكل شريط التنقل
## Navigation Bar Structure

### المكونات الأساسية:
### Basic Components:

1. **حاوية التنقل الرئيسية** (Nav Container)
   - تحتوي على الشعار والقوائم والإجراءات

2. **قائمة التنقل الرئيسية** (Main Navigation)
   - الصفحة الرئيسية
   - المنتجات
   - الاتصال
   - إلخ

3. **قائمة الملحقات المنسدلة** (Accessories Dropdown)
   - أهم مكون في النظام
   - يحتوي على العلامات التجارية والفئات والموديلات

4. **إجراءات المستخدم** (User Actions)
   - البحث
   - السلة
   - حساب المستخدم

## قائمة الملحقات المنسدلة
## Accessories Dropdown Menu

### الوظائف الأساسية:
### Core Functionality:

1. **عرض العلامات التجارية** (Brand Display)
   - يتم جلب العلامات من قاعدة البيانات
   - يدعم الأسماء العربية والإنجليزية
   - يعرض عدد الفئات لكل علامة

2. **القوائم الفرعية** (Submenus)
   - تظهر عند التمرير على العلامة التجارية
   - تحتوي على جميع الفئات لهذه العلامة
   - تتضمن رابط "جميع الفئات"

3. **الاستجابة للأجهزة** (Device Responsiveness)
   - سطح المكتب: قوائم منسدلة عادية
   - الموبايل: قوائم قابلة للنقر

### آلية عمل القوائم الفرعية:
### Submenu Operation Mechanism:

```
عند التمرير على علامة تجارية:
1. يتم جلب الفئات من قاعدة البيانات
2. تظهر القائمة الفرعية
3. يمكن التنقل إلى الفئة المرغوبة
4. إخفاء القائمة عند الخروج
```

## هيكل قاعدة البيانات
## Database Structure

### الجداول الرئيسية:
### Main Tables:

#### 1. جدول العلامات التجارية (brands table)
```sql
- id: معرف العلامة التجارية
- name: الاسم الإنجليزي
- name_ar: الاسم العربي
- status: حالة النشاط
- categories_count: عدد الفئات
```

#### 2. جدول الفئات (series table)
```sql
- id: معرف الفئة
- brand_id: معرف العلامة التجارية المرتبطة
- name: الاسم الإنجليزي
- name_ar: الاسم العربي
- status: حالة النشاط
```

#### 3. جدول الموديلات (models table)
```sql
- id: معرف الموديل
- series_id: معرف الفئة المرتبطة
- name: الاسم الإنجليزي
- name_ar: الاسم العربي
- status: حالة النشاط
```

### العلاقات بين الجداول:
### Table Relationships:

```
brands (1) ──── (many) series
    │                    │
    └──────── (many) ───┘
         models
```

## الملفات المرتبطة بالنظام
## Related System Files

### 1. inc/topBarNav.php
**الغرض:** الملف الرئيسي لشريط التنقل
**المحتوى:**
- HTML للهيكل الأساسي
- CSS للتصميم والاستجابة
- JavaScript للتفاعل والأحداث
- استعلامات قاعدة البيانات للعلامات التجارية

### 2. ajax/get_brand_categories.php
**الغرض:** جلب فئات العلامة التجارية
**الوظيفة:**
- يستقبل معرف العلامة التجارية
- يرجع الفئات المرتبطة بها
- يدعم الأسماء العربية والإنجليزية

### 3. ajax/get_category_models.php
**الغرض:** جلب موديلات الفئة
**الوظيفة:**
- يستقبل معرف الفئة
- يرجع الموديلات المرتبطة بها
- يدعم الأسماء العربية والإنجليزية

### 4. classes/DBConnection.php
**الغرض:** الاتصال بقاعدة البيانات
**المحتوى:**
- إعدادات الاتصال
- دوال مساعدة للاستعلامات

### 5. config.php
**الغرض:** إعدادات المشروع العامة
**المحتوى:**
- متغيرات البيئة
- إعدادات قاعدة البيانات

## آلية تحميل القوائم الفرعية
## Submenu Loading Mechanism

### العملية خطوة بخطوة:
### Step-by-Step Process:

1. **تحميل الصفحة** (Page Load)
   - يتم جلب جميع العلامات التجارية
   - يتم إنشاء هيكل HTML للقوائم

2. **التمرير على العلامة** (Brand Hover)
   - يتم إرسال AJAX لجلب الفئات
   - تظهر القائمة الفرعية

3. **التمرير على الفئة** (Category Hover)
   - يتم إرسال AJAX لجلب الموديلات
   - تظهر قائمة الموديلات

4. **النقر على الرابط** (Link Click)
   - يتم توجيه المستخدم لصفحة المنتجات
   - مع تمرير معرفات التصفية

## الاستجابة للأجهزة
## Device Responsiveness

### سطح المكتب (Desktop > 769px):
- قوائم منسدلة عند التمرير
- قوائم فرعية جانبية
- تجربة مستخدم كاملة

### التابلت (Tablet 769px - 1024px):
- قوائم مخفية
- إزالة قائمة الملحقات
- تصميم مبسط

### الموبايل (Mobile < 768px):
- قوائم قابلة للنقر
- قوائم فرعية تحت العنصر الأساسي
- تصميم عمودي

## Troubleshooting
